#  web_ticket
Sistema simples de gerenciamento de senhas para atendimento, com área do atendente, área do cliente e telão de chamadas.

---

##  Sobre o Projeto

O **web_ticket** é um sistema de geração e chamada de senhas feito em HTML, CSS e JavaScript.  
Ele simula o funcionamento de um painel de atendimento, permitindo:

- Geração de diferentes tipos de senhas  
- Visualização do painel no telão  
- Área separada para cliente e atendente  
- Chamadas de senha em tempo real (simulação)

O projeto é ideal para estudos ou implementação em pequenos ambientes de atendimento.

---

## Como utilizá-lo

Não é necessário quebrar a cabeça com esse sistema. Possui duas áreas principais:

- Área do cliente
- Área do atendente

Na área do atendente é possivel criar as senhas e entrão chamá-las quando acionado "Iniciar Atendimento".
Na área do cliente, é possível observar essas senhas em tempo real.
